# box-sizing   
<img src="https://github.com/haeyonghahn/TIL/blob/master/HTML-CSS/images/box-sizing.png" width="450px" height="300px" alt="box-sizing.png"></img>

* 왼쪽 그림 : border-box   
width의 길이는 contents의 width가 아니라 border의 길이로 잡히게 된다. height 또한 마찬가지이다.   
* 오른쪽 그림 : contents-box   
width, height의 길이는 contents의 길이가 된다.   
* default : contents-box;
