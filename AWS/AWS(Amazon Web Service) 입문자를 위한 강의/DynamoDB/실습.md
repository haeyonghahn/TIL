이러한 방식으로 사용할 때 하나씩 데이터를 넣을 수 밖에 없다. 다량의 데이터를 한번에 넣고 싶다면
- IoT -> DynamoDB
- S3 버켓에 파일 업로드 -> 람다 함수 발동 -> DynamoDB   

이러한 시나리오로 다량의 데이터를 한번에 넣을 수 있다. 여기선 람다 함수를 발동시켜 다량의 데이터를 DynamoDB에 입력해보자.
