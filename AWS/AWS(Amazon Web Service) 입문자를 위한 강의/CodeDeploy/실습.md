# 실습
EC2 인스턴스를 만들고 소스코드가 들어있는 S3 버킷에 접근하여 CodeDeploy agent라는 것을 설치한다.   
이후 CodeDeploy를 이용하여 간단한 웹 어플리케이션을 배포해보자. 필요한 역할들을 IAM에서 생성한다.
